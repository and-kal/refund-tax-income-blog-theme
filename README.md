run dev server with `

```
cd ./themes/cupper-hugo-theme/income-tax-refund-blog
hugo server --themesDir ../..
```


build /public/ folder with

```
cd ./themes/cupper-hugo-theme/income-tax-refund-blog
hugo --minify --destination ../../../public --themesDir ../..
```
